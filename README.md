## 🚀 Run Project

1️⃣ Clone project and access its folder:

```bash
$ git clone https://github.com/jtiagosantos/design-system.git
$ cd design-system
```

2️⃣ Install dependencies:

```bash
$ npm i
```

3️⃣ Start project:

```bash
$ npm run dev
```
